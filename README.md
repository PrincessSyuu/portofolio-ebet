# Elbert Student Portfolio

Created by student named Elbert from Tunas Bangsa Cakung High School


A simple **single-page student portfolio** website.

## Features

- Responsive layout (mobile + desktop)
- Tailwind CSS (via CDN)
- Scroll reveal animations (IntersectionObserver)
- Smooth scrolling and navbar scroll effect (glassmorphism)
- Contact section with a demo-only form handler

## Tech Stack

- HTML
- Tailwind CSS (CDN)
- Vanilla JavaScript

## How to Run

Since this project is static, you can open `index.html` directly in your browser.

### Recommended

- Open `index.html` in your browser.
- If you preview via VS Code, use a simple “Live Server” extension (optional but helpful).

## Customization

- Update content directly in `index.html` (sections: About, Excellence/Journey, Projects, Contact).
- Update profile image URL:
  - Find the `<img>` tag in the Hero section.
- Update contact links:
  - Email/GitHub/Instagram links are in the Contact section.
